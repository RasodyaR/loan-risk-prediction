# loan-risk-prediction
Loan risk prediction using logistic regression with data preprocessing, feature engineering and performance evaluation on the German credit dataset.

This project builds a machine learning model to classify loan applicants as risky or not risky using the German Credit dataset. It applies a complete pipeline including data preprocessing, feature encoding, scaling, model training and evaluation.

Key Features

* Data cleaning and handling missing values
* Binary, ordinal, and one-hot encoding techniques
* Feature scaling using StandardScaler
* Logistic Regression model with class weighting
* Adjustable classification threshold for better risk detection
* Model evaluation using accuracy, confusion matrix, and classification report
* Visualization with Seaborn heatmap

Tech Stack

* Python
* NumPy, Pandas
* Matplotlib, Seaborn
* Scikit-learn
